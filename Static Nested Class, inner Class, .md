---
* 정적 중첩 클래스
  ```Java
  class OuterClass {
    ...
    static class StaticNestedClass {
        ...
    }
  }
  ```
* 정적 중첩 클래스는 외부에서 생성이 가능하고 정적이기 때문에 OuterClass의 내부 변수에 접근이 불가능

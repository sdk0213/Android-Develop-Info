filter() 함수
===
* 
* ```java
  String[] objs = {"1 CIRCLE", 2 DIAMOND", "3 TRIANGLE", "4 DIAMOND", "5 CIRCLE", "6 HEXAGON"};
  Observable<String> source = Observable.fromArray(objs)
    .filter(obj -> obj.endsWith("CIRCLE"));
  source.subscribe(System.out::println);

  // result:
  // 1 CIRCLE
  // 5 CIRCLE
